# Prasanna Rani — Innovative AI/ML Portfolio

A modern React + Vite portfolio built for AI/ML, Data Science, and GenAI internship applications.

## Run locally
```bash
npm install
npm run dev
```

## Edit your details
Open `src/main.jsx` and update:
- email
- LinkedIn link
- resume link
- project live demo links if needed

## Deploy on Vercel
1. Upload this folder to GitHub
2. Go to Vercel
3. Import the GitHub repository
4. Click Deploy
